userinfo.email identifies which Google account the user is connecting and stores it as the integration key. calendar.readonly lists the user's calendars for sync selection and reads event data for RSVP handling. calendar.events.owned creates, updates, and deletes calendar events when meetings are scheduled, rescheduled, or cancelled through our platform. calendar.events.freebusy fetches event titles and times to show a user's availability to invitees picking a time slot. calendar.freebusy is a fallback for shared or read-only calendars where events.list is unavailable; it is a separate API endpoint not covered by calendar.events.freebusy.

For Google Meet: userinfo.email identifies the connected account. meetings.space.created programmatically creates Meet spaces when Google Meet is selected as the conference provider, generating the meeting link that is embedded in the calendar invite sent to all participants.
